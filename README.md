This repository contains example HTML5 demos used in the ITI Lab 2 exercises,This workspace contains two small HTML demos for learning DOM and media controls.

Files
- [video-player.html](video-player.html): A custom video player using the HTML5 `<video>` element with play/pause, seek, volume/mute, and playback speed controls. Requires the file `ECMADay3Part2Bonus.mp4` (referenced in the HTML) placed alongside the page.
- [textColorChanger.html](textColorChanger.html): A demo that changes the color of the page header using RGB and alpha sliders.

Quick usage
- Open the files in your browser (double-click or use a local server).
- For the video player, ensure `ECMADay3Part2Bonus.mp4` is present in the same folder.

Notes
- These are plain HTML/JS files with no build step.
- Tested in modern browsers (Chrome, Edge, Firefox).
